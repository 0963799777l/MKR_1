Цей репозиторій містить розв’язок Модульної контрольної роботи №1 (Варіант 1–1) з курсу «Інтелектуальний аналіз даних і підтримка прийняття рішень».
Мета роботи — продемонструвати розуміння:
Оцінювання якості алгоритмів машинного навчання
Перехресної перевірки (cross-validation, K-Fold)
Байєсівської класифікації (MAP, ризик, матриця втрат)
Обчислення Gain / Gain-ratio для ознаки «Куріння» в задачі класифікації
